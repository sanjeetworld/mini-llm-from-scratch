# Mini LLM From Scratch

A character-level mini language model implemented from scratch using PyTorch.  
The model learns sequential text patterns and generates coherent text.

## 🚀 Features

- Character-level tokenization  
- Vocabulary creation  
- Embedding + Linear language model  
- Training with Adam optimizer  
- Cross-entropy loss  
- Text generation  

## 🧠 Tech Stack

- Python  
- PyTorch  
- NumPy  
- NLP  

## 📊 Dataset

WikiText dataset used for training.
